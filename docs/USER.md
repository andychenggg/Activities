# 用户文档 — 算账工具 Pro

## 这能帮你解决什么痛点？
- **多人活动记账太麻烦**：快速创建活动，记录每个人的支出与权重，自动算出谁该付/谁该收。
- **怕别人看不到账本**：生成免登录的公开只读链接，分享给任何人，支持设置有效期自动失效。
- **担心数据安全**：密码存储为哈希，管理员与普通用户分权，公共链接仅可查看。

## 快速开始
1. **登录**：访问前端并使用分配的账号密码登录。管理员账号会自动进入“管理员模式”；普通用户进入工作台。
2. **创建活动**：在左侧列表点击创建，输入活动名称即可。
3. **添加人员与支出**：在活动详情页添加人员（可设置权重），录入支出，系统自动计算统计与最佳平账方案。
4. **分享账本（可选）**：在活动详情页设置有效天数并生成分享链接，复制后发送给需要查看的人；到期后自动失效。
5. **管理员功能**：在管理员模式下可创建新用户（角色为 user）或重置用户密码；管理员不能使用普通功能。

## 核心功能一览
- **活动管理**：创建/重命名/删除活动；查看人员与支出列表。
- **人员管理**：添加/修改/删除人员，支持权重（按份额分摊）。
- **支出管理**：录入、删除支出；多笔输入时可合并或分笔提交（一笔时直接提交）。
- **统计与平账**: 自动汇总支出与应付/应收，生成推荐转账方案。
- **模板（配置）**：保存/导入常用人员配置。
- **分享链接**：为活动生成免登录只读链接，设置有效期（1–365 天），过期自动失效。
- **认证与权限**：
  - 普通登录：cookie 会话。
  - 管理员：只能在管理员模式操作用户，无法使用普通功能。
  - 公共分享：只读，不能编辑。

## 常见操作
- **生成分享链接**：进入活动详情 → “生成共享查看链接” → 设定有效天数 → 生成并复制。
- **查看分享页面**：访问 `/?share=<token>`，可滚动查看完整页面，无需登录。
- **管理员创建用户**：管理员模式 → “创建新用户” → 输入用户名/两次密码 → 创建。
- **管理员重置密码**：管理员模式 → “重置密码” → 输入目标用户名与新密码（需二次确认）。

## 安全与隐私
- 密码使用 bcrypt 哈希存储。
- 管理员与普通用户分权；公共链接仅可查看。
- 分享链接自动过期，过期后无法访问。

## 常见问题
- **分享链接还能复用吗？** 可以，直到到期；生成新链接不会自动失效旧链接（可手动删除数据库记录）。
- **管理员看不到普通页面？** 设计如此，管理员登录即进入管理员模式，无法使用普通记账功能。
- **只能看到一笔支出时，为何不弹选择？** 只有一笔时系统自动提交，省去合并/分笔选择。
